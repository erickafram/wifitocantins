# MANUAL DO MOTORISTA - WIFI TOCANTINS TRANSPORTE

## 1. COMO O MOTORISTA DEVE SE CONECTAR

### Passo a passo

1. Desative os dados moveis do celular.
2. Conecte no Wi-Fi do onibus.
3. Abra o portal de acesso.
4. Toque em **"Motorista? Ative seu voucher"**.
5. Digite o **CPF cadastrado** ou o **codigo do voucher**.
6. Toque em **"Buscar Voucher"**.
7. Confira os dados encontrados.
8. Toque em **"Ativar Voucher Agora"**.
9. Aguarde a confirmacao da conexao.

### O que o motorista precisa saber

- O motorista nao precisa pagar.
- O acesso do motorista depende de voucher vinculado ao cadastro.
- O sistema aceita busca por CPF ou por codigo do voucher.
- O voucher so funciona conectado no Wi-Fi do onibus.
- Se o voucher estiver em uso em outro aparelho, o sistema vai bloquear a ativacao.
- Se existir intervalo minimo entre ativacoes, a tela informa o horario da proxima liberacao.

### Fala curta para orientar o motorista

> "Conecta no Wi-Fi, entra em Motorista, digita CPF ou voucher e toca em Ativar Voucher. Nao precisa pagar."

---

## 2. COMO O MOTORISTA DEVE EXPLICAR PARA O PASSAGEIRO

### Passo a passo do passageiro

1. Pedir para o passageiro desligar os dados moveis.
2. Pedir para conectar no Wi-Fi **TocantinsTransporteWiFi**.
3. Aguardar a tela do portal abrir.
4. Pedir para tocar em **"Conectar Agora"**.
5. Pedir para informar o telefone com DDD.
6. Pedir para tocar em **"GERAR QR CODE PIX"**.
7. Pedir para fazer o pagamento PIX.
8. Aguardar a confirmacao.
9. A internet sera liberada automaticamente.

### Regras importantes para o passageiro

- O passageiro precisa pagar para usar a internet.
- O pagamento deve ser feito enquanto o aparelho estiver conectado no Wi-Fi do onibus.
- Depois de copiar o PIX, o sistema pode liberar internet temporaria por 3 minutos para abrir o app do banco.
- Depois da confirmacao do pagamento, o acesso fica liberado por 12 horas.

### Fala curta para orientar o passageiro

> "Conecta no Wi-Fi, clica em Conectar Agora, informa seu telefone, gera o PIX e faz o pagamento. Assim que confirmar, a internet libera."

## 3. RESPOSTAS RAPIDAS PARA DUVIDAS COMUNS

| Pergunta | Resposta |
|---|---|
| "Motorista paga?" | "Nao. O motorista ativa com CPF ou voucher." |
| "Passageiro paga?" | "Sim. O acesso do passageiro e por PIX." |
| "Nao apareceu a tela" | "Confirme se esta no Wi-Fi do onibus e abra o portal novamente." |
| "Posso pagar fora do Wi-Fi?" | "Nao e o ideal. O aparelho precisa estar no Wi-Fi do onibus para o sistema identificar corretamente o dispositivo." |
| "Quanto tempo dura para o passageiro?" | "12 horas apos a confirmacao do PIX." |
| "Por que o motorista nao conseguiu ativar?" | "Pode estar em outro aparelho, fora do horario de reativacao ou com limite diario atingido." |
